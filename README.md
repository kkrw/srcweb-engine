## 開発環境

- Python3 をインストール
- node.js をインストール
- backend に移動して python -m venv venv
- frontend に移動して npm install

### backend 開発

- backend に移動する
- ./venv/Scripts/activate を実行する

### frontend 開発

- frontend に移動する
- npm run dev
